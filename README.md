# 🚀 AI Resume Builder

An intelligent web application that helps users create professional, ATS-friendly resumes in minutes using AI.

# 📌 Features
- 🧠 AI-powered resume content generation
- 📄 Multiple resume templates
- ⚡ Real-time preview
- 📥 Download resume as PDF
- 🎨 Modern and responsive UI (Bootstrap)
- 🔐 User-friendly form-based input
- 🧾 ATS-friendly formatting
- 🛠️ Tech Stack

## Frontend:

- HTML5
- CSS3
- Bootstrap 5
- JavaScript

## Backend:

- Python
- Django

## Other Tools:
